# Incident Response

Purpose: What to do if things go wrong in production.

## Scenario: Convergence Never Achieved
1. Check logs for agent timeouts
2. Verify rubric thresholds aren't too strict
3. Check for circular dissent patterns
4. Manual override procedure

## Scenario: Trust Scores Collapsing
1. Check for adversarial agent behavior
2. Verify regret marking is working
3. Consider resetting suspect agent memory
4. Emergency memory recovery procedure

## 1. Critique (Adversarial)
Purpose: Identify flaws, blind spots, risks
Example: "This ADR doesn't address backwards compatibility"
Used by: Skeptic role, Devil's Advocate

## 2. Amplification (Synergistic)
Purpose: Build upon ideas, add dimensions
Example: "Building on your GraphQL point, we could also..."
Used by: Creator role in polyadic, optional Builder role

## 3. Clarification (Socratic)
Purpose: Probe assumptions, request elaboration
Example: "What's the rationale for choosing REST over GraphQL?"
Used by: Any agent when information is insufficient

## 4. Adjudication (Decisive)
Purpose: Choose between options, declare convergence
Example: "Based on rubric scores, Version 3 is superior"
Used by: Moderator, Consensus detector
